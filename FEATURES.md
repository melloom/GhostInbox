- **Message Sorting** - Sort by newest or oldest

- **Poll Creation** - Create polls with multiple options
- **Poll Expiration** - Set expiration dates for polls

- **Response Templates** - Use templates for quick responses
- **Response Tracking** - Track all responses sent
